# Brilenko
Cascadian design method for interaction interfaces name by "Brilenko"

# Content 
Content is goal for interaction interfaces. Interaction interface exists only for content.

# User Persona
Interaction interfaces design for specific user persona.

> [!TIP]
> A rough profile of a hypothetical user is better than an undefined audience.

Example: "Anna, 28 years old, marketer, who uses the service to quickly find statistics for reports. She values ​​speed, accuracy, and the ability to quickly copy data. She doesn't have time to figure out the interface."
